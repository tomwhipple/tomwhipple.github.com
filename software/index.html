Perl Modules
  • Log.pm - provides an easy way to separate logging by levels. You can write code for five levels (by default) of verbosity and adjust the logging level at runtime. (pod) (code)
  • Cmd.pm - provides an easy way to run a script in a child thread, process the output, and automatically log it in parallel. Requires Log.pm (pod) (code)
  • SQLstatement.pm - allows SQL to be abstracted from perl in a separate file. This is useful for MVC models or simply to easily debug the embedded SQL. Requires Log.pm (pod) (code)
